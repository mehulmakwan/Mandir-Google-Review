# Mandir Review Dashboard

This repository is now set up as a standalone static dashboard for Vercel.

## What it does
- Serves the dashboard from the dashboard folder.
- Loads review data from a local JSON export file.
- Works as a static site with no backend required.

## Deploy on Vercel
1. Import this repository into Vercel.
2. Use the repository root as the project root.
3. Vercel will serve the static files automatically.

## Files
- dashboard/index.html
- dashboard/styles.css
- dashboard/js/app.js
- dashboard/js/analysis.js
- dashboard/js/charts.js
- dashboard/js/data.js
